HERO SECTION MEDIA INSTRUCTIONS
===============================

To enable the hero section, set `$showHero = true` in:
`resources/views/frontend/Minimal/games/list.blade.php`

FILE REQUIREMENTS:
------------------
1. Desktop Image: 
   - Rename to: hero-desktop.jpg
   - Size: ~1600–1920px wide
   - Aspect Ratio: 16:9 or 2:1
   - Max Size: <400 KB

2. Mobile Image:
   - Rename to: hero-mobile.jpg
   - Size: ~1080px tall
   - Aspect Ratio: 3:4 or 4:5
   - Max Size: <300 KB

3. Video (Optional - Replaces Desktop Image):
   - Rename to: hero.mp4
   - Resolution: 1920×1080 or 1280×720
   - Format: MP4 (H.264)
   - Duration: 10–15s loop (Muted)
   - Max Size: <8–10 MB
